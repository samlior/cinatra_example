# cinatra_example

This is a example project for [cinatra](https://github.com/qicosmos/cinatra)

## Build

```sh
git clone https://github.com/samlior/cinatra_example.git
cd cinatra_example
mkdir build && cd build
cmake -DCMAKE_BUILD_TYPE=Release -G Ninja ..
ninja
```
